## Dev Setup Requirements

-   `NodeJs 14`
-   `MongoDB`, `MongoDB Compass`
-   `VsCode`, `eslint`, `prettier`

## How to run App locally

-   `yarn` - to install dependencies
-   `yarn start` - to start react app at http://localhost:3000
